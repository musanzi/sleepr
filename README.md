# Sleepr

This project is part of my learning journey in microservices with NestJS.

## Goal

I am learning microservices in NestJS by building and exploring how services communicate, scale, and stay maintainable in a distributed system.

## Tech Stack

- NestJS
- TypeScript
- PNPM

## Project Setup

```bash
pnpm install
```

## Run the Project

```bash
# development
pnpm run start

# watch mode
pnpm run start:dev

# production mode
pnpm run start:prod
```

## Run Tests

```bash
pnpm run test
pnpm run test:e2e
pnpm run test:cov
```

## Notes

This repository is mainly for practice, experimentation, and understanding core microservice concepts in NestJS.
